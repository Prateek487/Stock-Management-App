
# Stock Management Application

It is a GUI based Python application which allows the user to store and manage information like quantiy, price, etc about different products. It provides the user an option to add and remove a product. It also provides user an option to change the value of the fields for a product and automatically adjust the respective fields accordingly. User can also search for a specific product and save the data into an excel file.
